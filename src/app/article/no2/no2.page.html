ionicの準備

公開日:2022年4月28日

執筆者:montsp

種類

ionic

ionicを始めるには、

・Windows(macを推奨)もしくはMacを搭載したPC

・Node.js

・git

Xcode(macのみ)

Android Studio

エディター(Visual Studio Codeが望ましい)

ionic CLI


Node.jsインストール

まず、nodejs.orgへ行きま

どちらかをインストールします。(最新版かLTS。)その後は、インストーラーの指示に従ってsインストールしてください。

ターミナルを開き、node -v　と入力し、バージョン番号が表示されたら成功です。


gitインストール(Xcodeをインストールする場合は、この手順は不要です。)

まず、git-scm.comへ行き、ダウンロードします。

node.jsと同様に、ターミナルを開き、git --version　と入力し、バージョン番号が表示されたら成功です。


Xcode(macのみ)インストール

Mac Appstoreを開いて、Xcodeと入力し、インストールします。


Android studioインストール

こちらにアクセスし、インストールします。

初期設定では、Standardを選ぶと完了です。

Visual Studio Codeのインストール

こちらにアクセスし、インストールします。

インストールが終わると、VSCodeを起動します。

ionicのサポート機能と、日本語をインストールします。

ionic CLIのインストール

ターミナルを開き、

npm install -g @ionic/cli

と入力し、エンターを押すと、@#%#%$^$%T%&とぐしゃぐしゃになってきます。

インストールが終わった気配が出たら、

ionic -v

と叩いて、バージョン番号が表示されたら、成功です。

これで準備完了ー😩
